    Keg Cop Bulk Loader:

This page will allow you to download your JSON configuration files. You may also upload a replacement file. With great power comes great responsibiility, so please be aware of the following::

  • If you upload a JSON file replacing either appconfig.json or flowconfig.json, you must immediately reset the controller with the button, not the function from the settings page!! If you wait, or use the reset in the application UI, your changes will be overwritten..
  • GUID changes are possible, but the impact on other systems has not been tested.
  • No "sanity checks" are done on this data. It is possible to render the controller inoperative and require re-flashing the firmware, losing your configuration.
  • The intent of this page is to allow you to download a backup copy of the JSON, or upload a complete replacement. This was not created for casual use, and this should not be necessary in your normal work with the application.
  • Any settings added to the JSON which are not used will be removed. Any settings not provided will have default values applied.
