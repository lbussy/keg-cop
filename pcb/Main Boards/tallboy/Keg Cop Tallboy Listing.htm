This is a printed circuit board design supporting Keg Cop.

The board provides the necessary component connections and circuitry for the ESP32 controller and connectors used in this project. This design is intended to support up to six flowmeters, as well as temperature control..

Please see the documentation for more information..

BOM

  • 2 x 0.1μF Ceramic Capacitor (C1, 3)
  • 2 x 1.0μF Ceramic Capacitor (C2, C4)
  • *1 x DS18B20 PCB-mount sensor (IC1)
  • 5 x 2.2kΩ 1/4W 5% Axial Resistor (R1-R5)
  • 2 x RJ45 8-pin 90° no shield PCM-mount jack
  • 1 x 2-pin Straight Male Header (POWER)
  • *1 x 3-pin Straight Male Header (ROOM)
  • 1 x 4-pin Straight Male Header (RELAY)

Additionally, you will need:

  • 1 x Lolin D32 v1.0.0
  • 2 x 16-Pin stacking male headers (almost always comes with the D32)
  • 2 x 16-Pin stacking female headers
  • *1 x Waterproof DS18B20 Temperature Sensor with lead

(*) - Optional items for use when a room temperature sensor is desired. Either a DS18B20 sensor with a lead is used, or a PCB-mount sensor is employed.
